# рҹҡҖ Quick Start

## Clone Repository

```bash
git clone https://github.com/your-username/scholarhub.git

cd scholarhub
```

## Install Dependencies

```bash
npm install

cd backend
npm install
```

---

## Environment Variables

### Frontend

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_API_URL=
VITE_SOCKET_URL=
```

### Backend

```env
PORT=5000

SUPABASE_URL=
SUPABASE_ANON_KEY=
```

---

## Run Frontend

```bash
npm run dev
```

## Run Backend

```bash
cd backend
npm run dev
```
